Εντολές Makefile:
make compile για παραγωγή των .class αρχείων		(javac *.java)
make execute για να τρέξει το πρόγραμμα				(java calculator)
make clean για να σβήσει τα .class αρχεία			(rm -f *.class *~)

Πολυψήφιοι με leading zeroes (πχ. 05, 007, 00019) δεν γίνονται αποδεκτοί από το πρόγραμμα.
Το πρόγραμμα δε δέχεται inputs που περιέχουν κενό ανάμεσα στα νούμερα (πχ. 5 + 3, αντί για 5+3).
Το πρόγραμμα λειτουργεί σωστά για όλα τα test cases που είχαν δωθεί απο συμφοιτητές στο piazza.
Η μορφή της γραμματικής που εκτέλεσα προσπαθεί να ομοιάσει στην LL(1) γραμματική που υπάρχει στις διαφάνειες του μαθήματος, τόσο σε μορφή, όσο και στις ονομασίες των κανόνων.
Το boolean flag, που περνιέται στις περισσότερες συναρτήσεις, σηματοδοτεί το αν το parsing που λαμβάνει μέρος εκείνη τη στιγμή, βρίσκεται μέσα(1) ή έξω(0) από παρένθεση.
